Things I build
for the joy of it.

Small interactive toys I prototype by pairing with AI — half design play, half code. Everything here runs right in your browser. Poke at them, break them, make a mess.

01

Floating Letters

Type anywhere and each letter drifts up like a balloon — a vibrant color, a little tilt, a gentle bob — crowding together at the ceiling. Hit clear to release them all and start fresh.

JavaScript · requestAnimationFrame · Interaction Design · Built with Claude

02

Liquid Type

Type to fill the screen with massive type, then sweep your cursor through it — the letters repel, swell, tilt, and shimmer cyan-to-magenta like they're suspended in liquid. Backspace deletes, Esc clears.

Canvas 2D · Spring physics · Typography · Built with Claude

Open full screen ↗ View source ↗
03

Constellation Builder

Click across the night sky to place stars — nearby ones thread themselves together with light, and clusters of a few bloom into their own named constellations. Hover to trace the connections.

Canvas 2D · Generative · Interaction Design · Built with Claude

Open full screen ↗ View source ↗
04

Coffee Table Book

A 3D book of my artwork you can actually open and leaf through — lift the cover, turn spread by spread in real perspective, then close it shut again. Use the ‹ › buttons to page through.

CSS 3D · Animation · Interaction Design · Built with Claude

Open full screen ↗ View source ↗
05

Wish You Were Here

Type any city and it develops a dithered postcard of the place — pulling the real weather and a real photo, then stamping and postmarking it like it actually traveled to reach you. Every card you make lands in the collection to leaf back through.

Canvas · Dithering · Open-Meteo + Wikipedia APIs · Built with Claude
